# Task 001 — sum_list

Implement `sum_list(values)` so that it returns the sum of a list of numbers.

- Input: a list of integers or floats (possibly empty).
- Output: the arithmetic sum; `0` for an empty list.

The seed implementation in `seed_solution.py` is intentionally naive. The
code-improver loop (Goal 02) proposes replacements scored against `tests.py`.
